2   Simics/Malta Overview

Simics/Malta models the Malta reference board from MIPS Technologies. The Malta board includes a MIPS 4Kc or 5Kc processor and additional devices. The 4Kc and 5Kc processors are set to run in little endian mode.

Simics/Malta loads the and executes the linux kernel directly without any bootloader (YAMON). The kernel includes a linked in initrd ramdisk with a few basic utilities, based on busybox. Since the bootloader normally sets up the PCI mappings, Simics/Malta cannot be equipped with PCI cards.
